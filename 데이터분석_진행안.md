# Orange3 데이터 분석 진행안 (교사용)

앱 사이드바의 '📚 분석 가이드'가 **학생용 조작 매뉴얼**이라면, 이 문서는 **교사용 진행안**입니다.
수업을 어떤 순서로 끌고 갈지, 각 단계에서 학생들이 실제로 무엇을 보게 되는지를 **삼성전자 실제 목업 데이터**로 미리 따라가 봅니다.

---

## 1. 분석 대상 데이터 요약

학생 화면에서 받는 통합 Excel의 시트 4개가 각각 다른 분석에 쓰입니다.

| 시트 | 내용 | 어울리는 분석 |
| --- | --- | --- |
| 과거3개년데이터 | 2023~2025 분기별 12행 | 시각화 · 의사결정나무 (첫 전략의 근거) |
| 월별실적보고서 | 2026_01부터 매달 추가 | 분기가 진행될수록 나무 재학습용 데이터 증가 |
| 시뮬레이션결과 | 전략 → 주가/실적 결합 기록 | "무엇이 주가를 움직였나" 복기 |
| 전체시장 | 40개 기업 최신 스냅숏 | K-평균 군집화 |

각 시트의 2~3행은 Orange3용 자료형/역할 헤더입니다. `마케팅_위기_등급`은 이미 **class(target)** 로 지정되어 있어 File 위젯이 자동 인식합니다(안 될 때만 Role을 수동 변경).

**위기 등급 산출 규칙(교사만 아는 정답지)** — 아래 4개 중 해당 개수로 결정:
재고율 ≥ 30% · 고객 이탈률 ≥ 15% · ROAS < 120% · 고객 만족도 < 70점 → **0개=안전, 1개=주의, 2개 이상=위험**.
의사결정나무 수업의 목표는 학생들이 이 규칙을 **데이터에서 스스로 역산**해내는 것입니다.

---

## 2. 수업 진행 흐름 (게임 루프와 연결)

```
[1차시] 데이터 열기 + 눈으로 관찰      → STEP 1~2 (File, Data Table, Box Plot)
[2차시] 의사결정나무로 위기 규칙 찾기   → STEP 3~4 (Tree, Tree Viewer, Test & Score)
[3차시] 첫 마케팅 전략 제출            → 교사: AI 평가 → 1분기 진행
[4차시~] 실적 보고서 재분석 → 전략 수정 → 분기 반복 (루프)
[마지막] 전체시장 군집화 + 시뮬레이션결과 복기 → 발표
```

- 1~2차시는 과거3개년데이터만 사용합니다. 분기를 아직 진행하지 않아도 됩니다.
- 4차시부터는 "내 전략이 데이터를 바꿨다"는 걸 확인하는 게 핵심 동기 부여입니다. 월별실적보고서 시트가 3행(1분기=3개월)씩 늘어난 것을 먼저 보여주세요.
- 군집화(STEP 5)는 별도 차시로 빼도 되고, 시장 이벤트가 산업군을 흔든 직후("왜 우리 산업만 떨어졌지?")에 넣으면 몰입도가 좋습니다.

## 3. 실전 예제 — 삼성전자 데이터로 전 과정 따라가기

아래 숫자는 실제 `data/삼성전자_초기분석데이터.csv` 값입니다. 시드가 고정되어 있어 수업에서도 동일하게 재현됩니다.

### STEP 1. 불러오기 확인 포인트

`File → Data Table` 연결 후 12행이 보이면 성공. 등급 분포를 먼저 세어보게 하세요.

> 삼성전자: **위험 6개 · 주의 5개 · 안전 1개** — "12분기 중 딱 한 번만 안전했던 회사"라는 스토리가 나옵니다.

### STEP 2. Box Plot으로 '아픈 지표' 찾기

`File → Box Plot`, Variable을 바꿔가며 Subgroups=`마케팅_위기_등급`으로 관찰합니다.

| 지표 | 삼성전자에서 보이는 것 | 판정 |
| --- | --- | --- |
| 고객 이탈률(%) | 12분기 중 7번이 15% 이상 (최고 24.7%) | 🔴 최다 발생 문제 |
| 고객 만족도(점) | 12분기 중 8번이 70점 미만 | 🔴 만성 문제 |
| ROAS(%) | 120% 미만은 2023_1Q(95%) 단 1회 | 🟢 광고 효율은 양호 |
| 재고율(%) | 30% 초과는 2024_4Q(31.5%) 단 1회 | 🟢 재고는 양호 |

→ 학생 결론(기대): **"우리 회사의 위기는 광고나 재고가 아니라 '고객이 떠나는 것'이다."**
→ 처방전 표와 연결: 이탈률 → 멤버십·재구매 쿠폰·AS 강화 방향의 전략이 나오면 성공.

유일한 '안전' 분기인 **2025_4Q**(이탈률 8.1%, 만족도 90점)를 반례로 보여주면 논리가 완성됩니다.

### STEP 3. 의사결정나무 읽기 — ⚠️ 반드시 연습용 통합 CSV 사용

**한 기업의 12분기만으로 학습시키면 안 됩니다.** 데이터가 12행뿐이라 나무가 우연히 등급과 맞아떨어진 엉뚱한 지표를 첫 갈림길로 잡습니다(실제 사례: 삼성전자는 'ESG 평판 ≤ 53'인 4개 분기가 우연히 전부 '주의'라서 ESG가 첫 갈림길이 됨 — 과적합의 좋은 토론거리).

그래서 의사결정나무 차시에는 통합 Excel의 **'전체기업3개년' 시트**(40개 기업 × 12분기 = 480행)를 **새 File 위젯**으로 열게 하세요. `File(전체기업3개년) → Tree → Tree Viewer`. (별도 배포 불필요 — 통합 분석 파일에 포함되어 있으며, 예비용 CSV는 `data/의사결정나무_연습용_전체기업_3개년.csv`)

- 480행에서는 첫 갈림길이 **고객 이탈률 ≈ 15**로 나오고, 그 아래로 **재고율 ≈ 30, ROAS ≈ 120, 만족도 ≈ 70**이 이어집니다 — 위기 등급 산출 규칙을 나무가 그대로 복원합니다.
- 나무 검증 활동: 우리 회사 최신 분기(삼성전자 2025_4Q: 재고 9.5 / ROAS 164 / 이탈 8.1 / 만족 90)를 손으로 갈림길에 넣어 '안전' 잎에 도착하는지 따라가기.
- 연습 파일 배포가 어려우면 차선책으로 `File → Select Columns`(Features를 전략 지표 4개만) → Tree. 엉뚱한 지표는 사라지지만 경계 숫자는 부정확합니다.

### STEP 4. 채점에서 예상되는 상황 (미리 알아두기)

`File(연습 파일) + Tree → Test & Score`(선 2개). 480행으로 학습하므로 **CA가 0.85~0.95 안팎으로 높게** 나옵니다. 여기서 File을 우리 회사 파일(12행)로 바꿔 다시 재면 0.5~0.8로 뚝 떨어집니다 — **"데이터가 많을수록 컴퓨터가 똑똑해진다"를 즉석에서 시연**할 수 있는 순간입니다.

- 분기를 진행해 월별실적보고서가 쌓이면 우리 회사 데이터의 CA도 오르는 것을 루프마다 확인 (데이터 축적의 가치를 체험).
- Tree 옆에 kNN을 같이 연결해 모델 대결을 붙이면 지루할 틈이 없습니다.

### STEP 5. 전체시장 군집화

File 시트를 `전체시장`으로 변경 → `k-Means(k=3)` → `Scatter Plot(Color=Cluster)`.

- 축 추천: X=`ROAS(%)`, Y=`고객 이탈률(%)` 또는 X=`매출액`, Y=`시장점유율` — 우량/평범/위기 그룹이 색으로 갈라집니다.
- 핵심 질문: **"우리 회사는 어느 클러스터인가? 잘나가는 클러스터와 어떤 숫자가 다른가?"**
- 심화: 산업군별 평균 등락률 차트(학생 화면)와 연결해 "클러스터 = 산업군인가, 아닌가?"를 토론.

## 4. 분기 루프에서의 반복 분석 (4차시~)

분기가 진행될 때마다 이 3가지를 루틴으로 돌리세요.

1. **월별실적보고서 재분석**: File 시트만 바꾸면 기존 워크플로우(Box Plot, Tree)가 그대로 재사용됩니다. "지난 분기 전략 이후 이탈률이 내려갔는가?"를 확인.
2. **나무 재학습**: 데이터가 3행씩 늘 때마다 Tree Viewer의 갈림길이 정답 규칙에 수렴하는지 관찰. Test & Score CA 상승 추이를 조별로 기록하게 하면 좋습니다.
3. **시뮬레이션결과 복기**: 전략 텍스트·AI 배수·시장 이슈·주가가 한 행에 묶여 있으므로, "AI 배수가 높았는데 주가가 떨어진 분기"를 찾아 시장 이벤트의 영향을 분리해 보는 토론이 가능합니다.

## 5. 산출물 · 평가 아이디어

- **분석 보고서 1장**: ① 우리 회사의 가장 아픈 지표(Box Plot 캡처) ② 나무가 찾은 규칙 ③ 그 근거로 세운 전략 — 전략 제출과 세트로 받으면 "데이터 근거 없는 전략"이 자연히 걸러집니다.
- **최종 발표**: 시뮬레이션결과 시트로 "우리 조 주가를 가장 크게 움직인 것은 전략이었나, 시장 이슈였나"를 검증해 발표.
- **역산 챌린지(심화)**: 위기 등급 산출 규칙 4가지를 나무만 보고 맞히는 조에게 보너스 — 정답은 이 문서 1절에 있습니다.
