# 융합형 주식 시뮬레이터 (교육용)

학생들이 기업 데이터를 분석해 마케팅 전략을 세우고, AI(Gemini) 평가와 거시 경제 이벤트에 따라 주가가 움직이는 것을 체험하는 교육용 웹 시뮬레이터입니다.

## 게임 진행 구조

1. **과거 데이터 분석**: 시뮬레이션은 2026년 1분기에 시작합니다. 학생들은 담당 기업의 과거 3개년(2023~2025) 재무/마케팅 데이터를 Orange3로 분석해 첫 전략을 세웁니다.
2. **마케팅 전략 제출 → AI 평가**: 교사가 조별 전략을 입력하면 AI가 시장 상황·최신 재무 지표와 함께 평가해 주가 등락률(배수)을 결정합니다.
3. **실적 보고서 생성**: 분기가 끝나면 마케팅 성과가 반영된 **월별 실적 보고서**(예: 2026_01, 2026_02, 2026_03)가 새로 생성됩니다. 좋은 전략은 매출·점유율·ROAS·만족도를 올리고 재고율·이탈률을 낮춥니다.
4. **다음 분기 전략 수립**: 학생들은 생성된 실적 보고서를 다시 분석해 다음 분기 전략을 세웁니다. 이 루프가 반복됩니다.

## 주요 기능

- **교사/학생 화면 분리**: 사이드바에서 모드를 전환합니다. 교사는 조별 마케팅 전략을 입력하고 분기를 진행시키며, 학생은 캔들스틱 차트와 재무 데이터를 확인합니다.
- **실제 기업 40개 풀**: 한국 8개 산업군 × 5개 기업. 4개 조가 각각 1개 기업을 선택해 운영하고, 나머지 36개는 마켓 이벤트에 따라 움직이는 시장 배경 역할을 합니다.
- **AI 마케팅 평가**: 학생 전략을 Gemini가 시장 상황·재무 지표와 함께 평가해 주가 등락률(0.75~1.25배)과 코멘트를 반환합니다.
- **마켓 이벤트**: 'AI 혁명', '글로벌 인플레이션' 등 14개 이벤트가 분기마다 랜덤 발생, 산업군별로 다른 가중치가 적용됩니다.
- **조별 주식 투자 (시드머니)**: 각 조는 1인 100만원씩 시드머니를 받고(1~3조 3명 = 300만원, 4조 4명 = 400만원), 40개 기업 중 원하는 주식을 삽니다. 매수/매도는 교사 화면의 '💰 주식 거래' 탭에서 교사가 대신 입력하며(오조작 방지), 같은 탭의 조별 포트폴리오 현황판에서 4개 조가 각각 무엇을 몇 주 들고 있는지 한눈에 볼 수 있습니다. 학생 화면의 전용 탭 '💰 조별 투자 성과 (주식 계좌)'에 조별 총자산 순위(🥇)와 계좌 상세(보유 종목별 매수단가·현재가·평가손익)가 표시되고, 모든 거래는 내역으로 기록되어 CSV로 내려받을 수 있습니다.
- **Orange3 호환 데이터**: 다운로드되는 CSV는 3줄 헤더(컬럼명/자료형/역할)가 붙어 있어 Orange3에 바로 넣어 군집화·의사결정나무 분석이 가능합니다. 분류 분석의 타겟 변수는 `마케팅_위기_등급`(안전/주의/위험)입니다.
- **시뮬레이션 결과 분석 CSV**: 분기가 진행될 때마다 '생성된 분기 실적 + 분기 말 주가 + 주가등락률 + AI평가배수 + 시장이슈 + 제출 전략 + AI코멘트'가 한 행으로 쌓입니다. 학생들이 "어떤 전략과 시장 상황이 실적과 주가를 움직였는가"를 데이터로 검증할 수 있습니다.
- **조별 누적 수익률 리더보드 + 레이스 차트**: 학생 화면 상단에 수익률 순위(🥇🥈🥉)와 함께, 4개 조의 주가를 시작=100으로 환산한 경쟁 그래프가 표시됩니다(110 = +10% 수익). 분기 점에 마우스를 올리면 수익률(%)과 그 분기의 시장 이슈가 보이고, 차트는 선이 그려지는 애니메이션과 함께 등장합니다.
- **전체 기간 주가 흐름**: 기업 카드의 캔들스틱 차트가 시뮬레이션 시작(2026년 1분기)부터 현재까지 진행된 모든 분기를 이어서 보여줍니다. 분기 단위 라인 차트도 따로 펼쳐볼 수 있습니다.
- **산업군별 평균 등락률 차트**: '전체 시장 동향'에서 이번 분기 이슈가 어떤 산업군을 웃고 울렸는지 막대 차트로 보여줍니다 (군집화 수업 연계).
- **기록 보존**: 시장 이슈 히스토리(학생 화면)와 분기별 전략/AI 평가 기록표(교사 화면)를 언제든 다시 볼 수 있습니다.

## 실행 방법

```bash
# 1. 라이브러리 설치
pip install -r requirements.txt

# 2. 40개 기업의 과거 3개년(2023~2025) 분석 데이터 생성 (최초 1회)
python generate_mock_data.py

# 3. API 키 설정: .streamlit/secrets.toml 파일에 아래 한 줄 입력
#    GEMINI_API_KEY = "발급받은-키"

# 4. 시뮬레이터 실행
streamlit run app.py
```

## 수업 운영 팁

- **진행 상태 자동 백업**: 분기가 넘어갈 때마다 `data/simulation_state.json`에 저장되므로 브라우저 새로고침이나 서버 재시작에도 진행 상황이 유지됩니다. 처음부터 다시 시작하려면 사이드바의 '전체 데이터 초기화' 버튼을 누르세요.
- **학생 데이터 다운로드**: 조별 기업 카드의 **통합 분석 파일(Excel)** 버튼 하나로 모든 데이터를 받습니다. 시트 구성:
  1. **과거3개년데이터** — 2023~2025 고정 데이터. 첫 전략 수립의 근거.
  2. **월별실적보고서** — 분기가 진행될 때마다 마케팅 성과가 반영되어 새로 생성되는 월 단위(2026_01~) 실적.
  3. **시뮬레이션결과** — 분기별 실적 요약 + 주가 + AI평가 + 시장이슈 + 전략 결합 데이터.
  4. **전체시장** — 40개 기업의 최신 실적·현재주가·등락률 (K-평균 군집화 실습용).
  5. **전체기업3개년** — 40개 기업 × 12분기 = 480행 (의사결정나무 실습용. 한 기업 12행만으로는 나무가 우연 상관에 과적합되므로 Tree/Test & Score는 반드시 이 시트로).
  Orange3의 File 위젯은 xlsx를 바로 읽으며 시트 선택이 가능합니다. 각 시트의 2~3행은 Orange3용 자료형/역할 헤더입니다.
- **분석 가이드 페이지**: 사이드바 '📚 분석 가이드' 모드에 Orange3 사용법이 STEP 1~5(Data/Visualize/Model/Evaluate/Unsupervised)로 정리되어 있습니다. 문해력 부담을 줄이기 위해 짧은 문장 + 카드형 시각 자료 + 위젯 연결 애니메이션으로 구성했습니다. 수업 시작 전에 학생들과 함께 보세요.
- **교사 기능 잠금**: '교사 화면(전략 적용)'과 '전체 데이터 초기화'는 PIN 인증 후에만 사용할 수 있습니다. 기본 PIN은 **2026**이며, `.streamlit/secrets.toml`에 `TEACHER_PIN = "원하는번호"` 한 줄을 추가하면 변경됩니다.
- **다크/라이트 모드**: 사이드바의 '🌞 라이트 모드' 토글로 화면 분위기를 전환할 수 있습니다.
- **분기와 달력**: 시뮬레이션은 2026년 1분기부터 시작하며 표준 분기 기준입니다(1분기=1~3월, 2분기=4~6월). 실적 보고서의 월 라벨(Year_Month)도 이 기준을 따릅니다.
- **배경 기업(36개)**: 주가는 마켓 이벤트에 따라 매 분기 변동하지만, 재무 데이터는 과거 마지막 실적(2025_4Q) 기준으로 고정됩니다.
- **마케팅_위기_등급 산출 기준** (의사결정나무 실습의 정답지): 재고율 30% 이상, 고객 이탈률 15% 이상, ROAS 120% 미만, 고객 만족도 70점 미만 — 해당 없음 '안전', 1개 '주의', 2개 이상 '위험'.
